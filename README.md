
# Fastify + TypeScript Template

Minimal Fastify template using TypeScript, pnpm, and sensible defaults for small services.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run in development (hot-reload)

```bash
npm run dev
```

3. Build and run

```bash
npm run build
npm start
```
